## [1.0.0] - TBD

### Added
- BibTeX picker for snacks.nvim
- Support for BibTeX, natbib, and BibLaTeX citation commands
- APA 7, Harvard, and Oxford citation format templates
- Frecency-based sorting with usage history
- LaTeX-to-Unicode conversion for accented characters
- Customizable keymaps and picker options
- Field-priority aware search and ranking
- Live preview for citation commands and formats
- Jump-to-source functionality for BibTeX entries
- Comprehensive configuration system
- Multi-locale support for citation formats
